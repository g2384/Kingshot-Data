Bear Hunt: How to earn higher rewards

Event Overview

  • Trap lasts 30 minutes
  • Scheduled and enabled by alliance leader and R4 members
  • No Governor Stamina used, no troop injuries, no troop loss, no resource cost
  • Rewards based on total damage

Preparation

  • Only allow top 10/15 players with highest power to start rally
  • Place rally initiators' castles close to the bear trap
  • Run event when most players are online
  • Save formations so that you can start or join rally very quickly
  • Troop formation:

    The best splits for each march is 10% infantry, 10% cavalry and 80% archers.

    If you don't have enough archers, you can also do 25:25:50, 15:15:70

  • Educate all members about this strategy

Start Rally

  • Use 3 strongest heroes to start rally
  • Kick joiners who use wrong heroes
  • Advanced: ask each joiner only send N troops, so that you have 15 joiners, with 15 recommended heros
